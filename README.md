# leetcode-javascript

个人使用 javascript 对 leetcode 上一些题目的解法

## 目录说明

- `/code` 存放解题方法 js 文件，按 leetcode 上的题目题号命名排列
- `/test` 存放配套的测试用例，使用 `jest` 进行单元测试
- `/analysis` 解题思路

## 运行

- `npm install` 安装依赖
- `npm test` 运行 `test` 文件夹下所有单元测试
- `npx jest ./test/文件` 运行某一个单独的测试用例，如 `npx jest ./test/1.test.js`
